What are HTML tags?

HTML tags are a visual way of representing elements in an HTML document. Their purpose is to provide an intuitive way of designing the layout and structure of an HTML web page.
Some common HTML tags include:

  • Header tags (<h1> - <h6>)
  • Paragraph tags (<p>)
  • Anchor tags (<a>)
  • Image tags (<img>)
  • List tags (<ul>, <ol> <li>)
Tag Name Syntax Description
Header <h1> - <h6> Used for headings
Paragraph <p> Used for paragraphs of text
Anchor <a> Used for hyperlinks
Image <img> Used to embed images
List <ul>, <ol>, <li> Used for lists
